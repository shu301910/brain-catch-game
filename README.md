# 脳トレキャッチゲーム

ボールをバーで跳ね返してブロックを壊し、モンスターを倒すブロック崩しゲーム。  
さらに、宇宙空間でモンスターに変身しながらブロックを避ける **COSMIC** モードも搭載。

---

## ゲームモード

| モード | 説明 |
|--------|------|
| 1P (1 BAR) | バー1個・ボール1個（白）のソロモード |
| 1P (2 BAR 2 Ball) | バー2個・ボール2個のソロモード |
| 2P | バー2個・ボール2個の2人対戦モード |
| COSMIC | 宇宙空間で降ってくるブロックを避けつつ、色ブロックで変身して能力を使うスコアアタックモード |

---

## 操作方法

### 1P (1 BAR)
| キー | 操作 |
|------|------|
| ← → | バー移動 |
| W / ↑ | ビーム発射 |

### 1P (2 BAR 2 Ball) / 2P
| キー | 操作 |
|------|------|
| ← → | 赤バー移動 |
| A D | 青バー移動 |
| W | 赤バービーム発射 |
| ↑ | 青バービーム発射 |

### COSMIC
| キー | 操作 |
|------|------|
| ← → / A D | 左右移動 |
| ↑ ↓ / W S | 上下移動 |
| SPACE | 現在の変身モンスターの能力を発動（ストック1消費） |

---

## ブロック崩しモード（1P / 2P）

### 特徴

- **カラーマッチ**: 赤ブロックは赤ボール、青ブロックは青ボールで壊す
- **プレイヤービーム**: バーからビームを発射してブロックを壊せる（色に関係なく破壊可能）
- **コンボ**: ボールが連続でブロックに当たると連鎖発生。スコアボーナス・ダメージ倍率・派手なエフェクト
- **モンスター**: 5種類のモンスターがそれぞれ特殊攻撃を持つ
- **2体モード**: 5体倒すと敵が2体同時に出現
- **ラスボス**: 9体倒すと巨大なラスボスが登場
- **ハイスコア**: モードごとにトップ10を記録

### モンスター一覧

| モンスター | 特殊攻撃 |
|-----------|----------|
| 🔴 Red | ボール速度アップ |
| 🔵 Blue | ボールが下降時に加速 |
| 🟡 Yellow | ボールを染色・光ビーム攻撃 |
| 🟢 Green | 操作が反転 |
| ⚫ Dark | バーの移動速度ダウン |

### ブロック一覧

| ブロック | 説明 |
|---------|------|
| 赤・青 | 対応色のボール・ビームで破壊。壊すと敵にダメージ |
| 紫 | 赤と青の両方のボール、またはビームで破壊。壊すとHPが回復 |
| 白（回復） | 壊すとHPが回復 |
| 無敵 | ビームを2秒当て続けると1ダメージ（3回で破壊）・35秒で自然消滅。破壊で無敵＆攻撃力アップが発動 |
| 金 | 5回当てると破壊・15秒で自然消滅。破壊するとビーム発射権を獲得 |

### コンボシステム

ボールがバーで跳ね返ってからブロックに連続で当てるとコンボが発生する。

#### コンボの仕組み

- バーで跳ね返ったあとブロックに当てるたびにコンボ数が +1
- バーに戻るとコンボ数はリセット
- どの種類のブロックでも、当たればカウントされる（壊さなくてもOK）

#### コンボの効果

| 効果 | 内容 |
|------|------|
| スコアボーナス | 2連鎖目から `コンボ数 × 5` ポイントが追加加算 |
| ダメージ倍率 | 赤・青ブロックを壊して敵を攻撃する時、威力アップ（最大 ×2.5）<br>※無敵状態（攻撃力アップ）発動中は適用されない |
| 視覚エフェクト | 2連鎖以上で「COMBO x N」テキストと光リング<br>4連鎖以上ではカラフルな粒子が散る派手な演出に |

コンボ数が増えるほど色が熱を帯びていく（白 → 金 → オレンジ → 赤）。

## COSMICモード

宇宙空間で、上から降ってくるブロックを避けながら色ブロックに触れてモンスターに変身し、能力でブロックを壊してスコアを稼ぐ。

### 特徴

- 上下左右に自由に移動できる回避シューティング風
- 時間経過で岩ブロックの落下速度と出現頻度が上昇
- 色ブロックに触れると対応するモンスターに変身
- ストックは最大3個まで貯められ、SPACEキーで能力発動
- 生存時間でも毎秒スコアが入る

### ブロック一覧（COSMIC）

| ブロック | 効果 |
|---------|------|
| ⬜ 白 | 害なし。通過用 |
| 🟫 岩 | 当たるとHP-1 |
| 🟣 紫 | 当たるとHP+1（最大9まで） |
| 🩵 シアン | 一定時間プレイヤーの移動速度UP |
| 🟡 金 | 一定時間、岩のダメージを無効化（シールド） |
| 🔴🔵🟡🟢⚫ 色ブロック | 触れると対応モンスターに変身しストック+1（ゲーム開始5秒後から出現） |

### モンスター能力（SPACEで発動）

| モンスター | 能力 |
|-----------|------|
| 🔴 Red | 爆弾＋プレイヤー周囲で連続爆発（爆竹）。範囲内のブロックを破壊 |
| 🔵 Blue | 上方向の水流を発生させ、ブロックを上に押し戻す |
| 🟡 Yellow | ランダムな位置に雷を3発落とす |
| 🟢 Green | 前方に風の渦を放ち、ブロックを巻き込んで破壊 |
| ⚫ Dark | 周囲のブロックをスロー化。岩は2秒で破壊 |

### スコア

- 生存ボーナス：毎秒 +5
- ブロック破壊：1個 +10

## セットアップ

### 必要環境
- Python 3.12以上
- pygame

### インストール
```bash
git clone https://github.com/shu301910/brain-catch-game.git
cd brain-catch-game
python -m venv venv
venv\Scripts\activate  # Windowsの場合
pip install -r requirements.txt
```

### 実行
```bash
python main.py
```

## ハイスコア

各モードのスコアは別ファイルに保存される。

| モード | ファイル |
|--------|---------|
| 1P (1 BAR) | `score_solo1.json` |
| 1P (2 BAR 2 Ball) | `score_solo2.json` |
| 2P | `score_duo.json` |
| COSMIC | `score_cosmic.json` |

各モードでトップ10まで記録され、ゲームオーバー画面ではトップ3が表示される。
