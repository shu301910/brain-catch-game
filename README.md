# 🧠 脳トレキャッチゲーム

ボールをバーで跳ね返してブロックを壊し、モンスターを倒すブロック崩しゲームです。

---

## 🎮 ゲームモード

| モード | 説明 |
|--------|------|
| 1P (1 BAR) | バー1個・ボール1個（白）のソロモード |
| 1P (2 BAR 2 Ball) | バー2個・ボール2個のソロモード |
| 2P | バー2個・ボール2個の2人対戦モード |

---

## 🕹️ 操作方法

### 1P (1 BAR)
| キー | 操作 |
|------|------|
| ← → | バー移動 |
| W / ↑ | ビーム発射 |

### 1P (2 BAR 2 Ball) / 2P
| キー | 操作 |
|------|------|
| ← → | 赤バー移動 |
| A D | 青バー移動 |
| W | 赤バービーム発射 |
| ↑ | 青バービーム発射 |

---

## ✨ ゲームの特徴

- **カラーマッチ**: 赤ブロックは赤ボール、青ブロックは青ボールで壊す
- **プレイヤービーム**: バーからビームを発射してブロックを壊せる（色に関係なく破壊可能）
- **モンスター**: 5種類のモンスターがそれぞれ特殊攻撃を持つ
- **2体モード**: 5体倒すと敵が2体同時に出現
- **ラスボス**: 9体倒すと巨大なラスボスが登場
- **ハイスコア**: モードごとにトップ10を記録

---

## 👾 モンスター一覧

| モンスター | 特殊攻撃 |
|-----------|----------|
| 🔴 Red | ボール速度アップ |
| 🔵 Blue | ボールが下降時に加速 |
| 🟡 Yellow | ボールを染色・光ビーム攻撃 |
| 🟢 Green | 操作が反転 |
| ⚫ Dark | バーの移動速度ダウン |

---

## 🧱 ブロック一覧

| ブロック | 説明 |
|---------|------|
| 赤・青 | 対応色のボールまたはビームで破壊 |
| 紫 | 赤と青の両方のボール、またはビームで破壊 |
| 白（回復） | 壊すとHPが回復 |
| 無敵 | ビームを2秒当て続けると1ダメージ（3回で破壊）・35秒で自然消滅 |
| 金 | 5回当てると破壊・15秒で自然消滅 |

## 🚀 セットアップ

### 必要環境
- Python 3.12以上
- pygame

### インストール
```bash
git clone https://github.com/shu301910/brain-catch-game.git
cd brain-catch-game
python -m venv venv
venv\Scripts\activate  # Windowsの場合
pip install -r requirements.txt
```

### 実行
```bash
python main.py
```
